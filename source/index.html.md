# Introduction

Welcome to the SimpleFundraiser API, and glad to have you on board!

SimpleFundraiser provides a simple and powerful REST API.

All product capabilites are supported through **five core resources**:

  1. **Account**
  2. **Fundraiser**
  3. **Prize**
  4. **Prize Program**
  5. **Seller**
